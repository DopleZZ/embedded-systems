# Fitocube Frontend

Фронтенд приложение для платформы Fitocube на React.

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:3000`

## Сборка

```bash
npm run build
```

## Структура проекта

- `src/pages/` - страницы приложения
- `src/components/` - переиспользуемые компоненты
- `src/api/` - API клиент для работы с бэкендом

## API

Приложение использует следующие эндпоинты:
- `GET /v1/plants/{plantId}` - получение растения по ID
- `GET /v1/plants/by-owner?ownerId={id}` - получение списка растений пользователя

По умолчанию используется `ownerId=1`. Можно изменить на главной странице.
